## Syntix 是什么？

Syntix 是一款 Chrome 浏览器扩展插件，帮助自媒体创作者将**一篇文章同时发布到多个内容平台**。你只需在 Syntix 中编辑或导入文章内容，勾选目标平台，点击发布，Syntix 会自动将文章填充到各平台的发布页面。

**已上架 Chrome 网上应用店**，安装即用，无需额外配置。

### 智能授权管理
通过插件内置的授权流程，一键登录各平台账号。授权后发布时自动携带登录状态，无需反复登录。

### 侧边栏工作台
Syntix 以浏览器侧边栏为核心操作入口，不占用网页空间，随时呼出、随时发布：
- **工作台首页**：文章统计一目了然
- **文章管理**：查看、搜索已通过审核的文章
- **发布管理**：勾选平台，批量或逐个发布
- **授权管理**：管理各平台授权状态

### 亮色 / 暗色主题
支持浅色和深色两种界面主题，跟随你的偏好。

## 如何使用？

### 第一步：安装
前往 Chrome 网上应用店搜索「Syntix」，或直接访问商店链接安装。

### 第二步：登录
点击浏览器工具栏的 Syntix 图标，在侧边栏中登录你的 Syntix 账号。

### 第三步：授权平台
进入「授权账号」页面，选择你需要发布的平台，按提示完成平台登录授权。授权后插件会自动保存登录状态。

### 第四步：选择文章并发布
1. 在工作台点击「文章列表」，选择已通过审核的文章
2. 点击文章旁的「发布」按钮
3. 在发布页面勾选目标平台（支持全选）
4. 点击「批量发布」或对单个平台点击「发布」

### 第五步：查看结果
发布完成后，每个平台会显示成功或失败状态。如果某个平台发布失败，可以点击「重试」单独重新发布。

## 常见问题

### Q：发布时提示需要登录怎么办？
部分平台的登录状态可能过期。请前往「授权账号」页面，重新对该平台进行授权，然后重试发布。

### Q：支持哪些浏览器？
目前支持 Chrome 和 Edge 等基于 Chromium 的浏览器。Firefox 版本也在开发中。

### Q：文章内容会经过 Syntix 服务器吗？
不会。文章内容仅在你的浏览器与目标平台之间直接传输，Syntix 不会中转或存储你的文章正文。

### Q：我的平台账号密码安全吗？
Syntix 绝不读取或存储你的平台账号密码。授权过程仅提取登录 Cookie 用于识别登录状态，且仅保存在本地浏览器中。

### Q：可以在 Mac / Windows / Linux 上使用吗？
可以。只要你的浏览器支持 Chrome 扩展（Manifest V3），均可正常使用 Syntix。
