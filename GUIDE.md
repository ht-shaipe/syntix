## 一、安装插件

### 从 Chrome 网上应用店安装（推荐）

1. 打开 Chrome 浏览器，访问 Chrome 网上应用店
2. 搜索「Syntix」
3. 点击「添加到 Chrome」
4. 等待安装完成，工具栏出现 Syntix 图标

> 也支持 Edge、Arc、Brave 等 Chromium 内核浏览器。

### 固定到工具栏

安装后图标可能被折叠，建议固定：

1. 点击工具栏右侧的 **拼图图标**（扩展程序）
2. 找到 **Syntix**
3. 点击右侧的 **图钉按钮** 将其固定

> 详细的离线安装方式请参考 [安装指南](./INSTALL.md)。

## 二、登录账号

1. 点击工具栏上的 Syntix 图标，打开侧边栏面板
2. 首次使用会显示登录界面
3. 输入你的账号和密码，点击「登录」
4. 登录成功后自动进入工作台首页

**说明：**
- 登录状态会自动保存在浏览器中，关闭浏览器后再打开无需重新登录
- 如需切换账号，点击右上角头像区域，选择退出登录

---

## 三、界面导览

Syntix 的核心操作区在**浏览器侧边栏**中，包含以下页面：

| 页面 | 入口 | 用途 |
|------|------|------|
| **工作台** | 首页 | 查看文章统计数据（总数 + 各状态分布） |
| **文章列表** | 左侧菜单 | 浏览已通过审核的可发布文章 |
| **授权账号** | 左侧菜单 | 管理各平台的授权状态 |
| **使用教程** | 左侧菜单 | 插件内置的操作指引 |
| **设置** | 左侧菜单 | 主题、账号等配置 |

侧边栏可以通过以下方式打开：
- 点击工具栏上的 Syntix 图标
- 使用浏览器侧边栏功能（Chrome: 点击侧边栏图标后选择 Syntix）

## 四、授权平台

发布文章前，需要先完成目标平台的授权，让 Syntix 获取登录状态。

### 授权流程

1. 在侧边栏点击「**授权账号**」进入授权管理页面
2. 页面会展示所有支持的平台（共 22 个）及当前授权状态
3. 找到你要发布的平台，点击「**授权**」按钮
4. 浏览器会自动打开该平台的登录页面
5. 在打开的页面中**正常完成登录**（输入账号密码或扫码登录）
6. 登录成功后，Syntix 会自动检测到登录状态并提示授权成功
7. 回到 Syntix 侧边栏，该平台会显示为「**已授权**」

### 授权状态说明

| 状态 | 含义 |
|------|------|
| 已授权 | 登录有效，可以发布 |
| 未授权 | 尚未完成登录 |
| 已过期 | 登录已失效，需要重新授权 |

### 注意事项

- 授权信息（登录 Cookie）仅保存在你的浏览器本地，不会上传到外部服务器
- 部分平台的登录状态有效期较短，可能需要定期重新授权
- 如果发布时提示「未授权」或「登录过期」，请回到此页面重新授权

## 五、发布文章

### 第一步：选择文章

1. 在侧边栏点击「**文章列表**」
2. 页面展示所有**已通过审核**的文章（只有通过审核的文章才能发布）
3. 你可以通过搜索框搜索文章标题
4. 点击文章右侧的「**发布**」按钮，进入发布页面

### 第二步：选择平台

进入发布页面后：

1. 顶部显示文章标题和作者信息
2. 下方列出所有**已授权的平台**
3. 默认已勾选全部已授权平台
4. 你可以：
   - 点击「**全选**」/ 取消全选来批量操作
   - 单独勾选或取消某个平台

### 第三步：发布

**批量发布：**
1. 确认勾选的平台无误
2. 点击右上角的「**批量发布 (N)**」按钮
3. 所有选中的平台会并行发布
4. 每个平台旁边会实时显示「发布中 → 成功/失败」

**单平台发布：**
1. 点击某个平台右侧的「**发布**」按钮
2. 该平台单独开始发布
3. 完成后显示结果

### 发布机制说明

Syntix 发布时会自动选择最优方式：
- **优先 API 发布**：直接调用平台接口，将文章保存为草稿，速度快
- **自动降级 DOM 模式**：如果 API 发布失败，自动切换到模拟填写发布页面的方式
- 两种方式都会将文章保存为**平台草稿**，你可以在平台编辑器中做最终调整后再发布

### 图片自动上传

文章中的图片会被自动处理：
- 支持自动下载远程图片并上传到目标平台的图床（支持 18 个平台）
- 确保发布后图片能正常显示
- 不支持图片上传的平台会保留原始图片链接

## 六、查看发布结果

发布完成后，发布页面底部会显示**结果汇总**：

```
✓ 知乎 — 成功
✗ 百家号 — 失败：未授权
✓ CSDN — 成功
```

- **成功**：文章已保存为该平台草稿，前往平台编辑器查看和发布
- **失败**：会显示具体原因（如「未授权」「连接失败」等）
- **重试**：失败的平台可以点击「**重试**」按钮单独重新发布
- **登录后重试**：如果失败原因是登录过期，会提示「登录后重试」，点击可跳转到授权页面

## 七、主题切换

Syntix 支持亮色和暗色两种界面主题：

1. 点击侧边栏右上角的**主题切换**图标（太阳/月亮）
2. 即可在亮色和暗色之间切换
3. 选择会自动记住，下次打开保持一致

## 九、常见问题

### Q：文章列表里没有文章？

文章列表只显示**已通过审核**的文章。如果列表为空，可能的原因：
- 你的文章还在审核中
- 文章被拒绝或还在草稿状态
- 请联系管理员确认文章审核状态

### Q：发布时提示「未授权」？

说明该平台的登录状态已过期或未完成授权。请前往「授权账号」页面，找到对应平台重新授权，然后回到发布页点击「重试」。

### Q：发布失败怎么办？

1. 查看失败提示信息
2. 如果是「未授权」→ 去授权页面重新授权
3. 如果是「连接失败」→ 等待片刻后点击「重试」
4. 如果是其他错误 → 截图反馈给管理员

### Q：文章发布到了哪里？

文章会保存为各平台的**草稿**，不会直接公开发布。你需要登录各平台，在草稿箱中检查内容、调整排版后手动发布。

### Q：支持哪些浏览器？

- Chrome、Edge、Arc、Brave 等 Chromium 内核浏览器
- Firefox 版本开发中

### Q：我的账号安全吗？

- Syntix 不读取也不存储你的平台账号密码
- 授权仅提取登录 Cookie 用于识别登录状态，保存在浏览器本地
- 文章内容在你的浏览器和目标平台之间直接传输，不经过第三方中转

### Q：Mac / Windows / Linux 都能用吗？

可以。只要你的浏览器支持 Chrome 扩展（Manifest V3），Syntix 都能正常使用。

### Q：发布速度慢？

- 批量发布时各平台并行处理，通常几秒内完成
- 如果某个平台响应慢，可能是因为该平台的服务器延迟，不影响其他平台的发布

## 十、小技巧

### 技巧 1：发布前检查授权状态

发布前先去「授权账号」页面确认所有目标平台都是「已授权」状态，可以避免发布中途失败。

### 技巧 2：批量发布 + 失败重试

遇到部分平台失败时不必全部重发，只需对失败的平台逐个点击「重试」即可。

### 技巧 3：从授权页面直接发布

在「授权账号」页面，点击某个已授权平台可以直接跳转到发布页，且自动选中该平台。

### 技巧 4：利用暗色模式

如果你经常在夜间创作，切换到暗色模式可以减轻视觉疲劳。
