# Stash 收藏助手 — 隐私政策

**生效日期**：2026-07-18
**适用范围**：Chrome 扩展「Stash 收藏助手」（以下简称「本扩展」）

本政策说明本扩展如何收集、使用、存储与保护你的信息。使用本扩展即表示你同意以下条款。

---

## 1. 我们收集的信息

本扩展遵循**本地优先（local-first）**原则，仅在为你提供功能所必需时处理数据。

### 1.1 本地存储（默认，无需登录）
当你收藏网页或创建笔记时，以下信息会保存在你浏览器的 **IndexedDB** 本地数据库中：
- 收藏对象的 URL、标题、描述；
- 内容类型（仓库 / 文章 / 视频 / 商品 / 动态 / 图片 / 书签 / 笔记）；
- 你添加的标签、分类；
- 笔记正文；
- 代码仓库的 stars / forks、编程语言等公开元数据（来自代码托管平台的公开 API）；
- 收藏 / 更新时间。

这些数据**仅存储于你的设备**，不会自动上传，除非你主动开启云端同步。

### 1.2 云端同步（可选，需主动开启）
当你登录账号并开启「云端自动备份」后，上述收藏数据会通过 HTTPS 加密传输至我们的云服务（`https://hub.htui.cc`）以便多设备同步。云端仅存储功能所需的最小字段（与本地一致），**不会**额外记录你的浏览历史。

### 1.3 账号信息
使用云端功能时，我们会收集你注册的**用户名与密码（以哈希形式存储，不可还原明文）**。我们**不收集**真实姓名、手机号、邮箱等强制个人信息；如你自愿提供，仅用于必要的账号与找回用途。

### 1.4 我们**不收集**
- 你不曾主动收藏的网页内容或浏览历史；
- 表单输入、键盘记录等任何与收藏无关的行为数据；
- 用于广告或跨站追踪的标识符。

---

## 2. 我们如何使用信息

- 在你的设备上展示、搜索、分类与管理你的收藏；
- 在你主动点击「刷新」时，向代码托管平台公开 API 请求仓库的 stars / forks 并展示；
- 在你开启云端同步时，备份与恢复你的收藏以实现多设备一致；
- 提供导入 / 导出功能，让你随时带走自己的数据。

我们**不会**将你的任何数据用于广告、画像或出售给第三方。

---

## 3. 数据共享与披露

- 我们**不与任何第三方共享、出售或交换**你的个人数据。
- 代码仓库的 stars / forks 来自 GitHub / Gitee 等平台的**公开 API**，不属于你的私密信息。
- 仅在法律要求或为保护本扩展与用户安全所必需时，才可能在法律程序下披露信息。

---

## 4. 数据存储与安全

- 本地数据存放于浏览器 IndexedDB，受浏览器沙箱保护；
- 云端传输使用 **HTTPS（TLS）** 加密；
- 密码以**哈希**形式存储，服务端不保留明文；
- 云服务访问需凭账号令牌（token），且遵循最小权限原则。

---

## 5. 你的权利与控制

你可以随时：
- **清空本地收藏**：在扩展「设置 → 数据管理 → 清空所有收藏」；
- **导出 / 导入**：将全部收藏导出为 JSON 文件备份或迁移；
- **关闭云端同步**：在「设置 → 数据同步」关闭自动备份；
- **删除云端数据**：退出账号后，云端对应收藏将被移除（你也可联系我们删除账号及全部数据）。

---

## 6. 权限用途说明

| 权限 | 用途 |
|------|------|
| `activeTab` / `tabs` | 在你主动收藏时读取当前页 URL / 标题 |
| `storage` | 保存主题、同步开关等设置 |
| `contextMenus` | 提供右键「收藏到 Stash」入口 |
| `notifications` | 收藏 / 同步结果的轻提示 |
| 代码平台 host 权限 | 仅在你点击「刷新」时获取仓库公开 stars/forks |
| `hub.htui.cc` host 权限 | 云端同步 API 地址 |

---

## 7. 儿童隐私

本扩展不面向 13 岁以下儿童，我们也不会故意收集其信息。

---

## 8. 政策变更

如本政策变更，我们会在扩展更新说明或官网公示。继续使用即视为接受更新后的政策。

---
